# 軟體架構圖紀錄方式

## Context and Problem Statement

軟體架構圖如何用純文字方式展示的最佳方式或解決方案？讓一個企業可以用共同標準來展示，避免使用PPTX或是其他無法版本控制的binary檔案

## Considered Options

* PlantUML
* Mermaind
* C4-PlantUML+Mermaid

## Decision Outcome

Chosen option: "C4-PlantUML+Mermaid", because C4 Model 支援更完整 (官方 C4-PlantUML 函式庫)，混合使用：不同層級使用不同工具
